# info-cli

__Command line tool for University of Evry__

```
Usage:
  info-evry [command]

Available Commands:
  completion  generate the autocompletion script for the specified shell
  contact     Search for a teacher's contact information
  help        Help about any command
  timetable   Get a link to timetable image

Flags:
  -h, --help   help for info-evry

Use "info-evry [command] --help" for more information about a command.
```
